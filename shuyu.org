\chapter*{术语}


- 上将、上帅 :: 将或帅上移一步；  
- 下二路 :: 从棋盘底线往上的第2条或者第9条横线。
- 九宫 :: 将帅可以活动的范围称之为九宫，正好有9个点位。
- 仕相全 :: 在炮兵残棋中提及较多，这是对红方而言的，指双士双相俱全，但对于绝大多数情况，缺一个士也无妨。
- 低兵 :: 指处于对方下二路的兵；
- 低车 :: 不可以平车打将的车位；
- 八角马 :: 指处于对方士角位的马，并且把对方的老将困在九宫的对角点上；
- 出将、出帅 :: 将或帅从中路平移一步到肋线；
- 单士象 :: 士象各缺一个，即一个士一个象；
- 单缺士 :: 只缺一个士，即一个士两个象；
- 单缺象 :: 只缺一个象，即一个象两个士；
- 占中  :: 用将帅或者车占领中路。
- 占肋  :: 用将帅、车或者炮占领肋线。
- 士象全 :: 这是对黑方而言的，指双士双相俱全。
- 布置线、宫顶线 :: 从棋盘底线开始数的第3条或者第8条横线，称之为布置线是因为开局阶段马、炮等子力通常密集布置在这一条横线上，称之为宫顶线是因为这条横线正好通过九宫顶部。
- 底兵 :: 指处于对方底线的兵；
- 底线 :: 从棋盘底线开始数的第1条或第10条横线。
- 立马车 :: 指处于对方3卒或7卒位的马，和车配合对于黑方暴露在肋线的将所形成的杀棋，特别是在黑方有中士的情况下。的时候马处于对方的中卒位也可以构成类似的杀棋，一并称作立马车；
- 花心、宫心 :: 九宫的中心，即下二路和中线的交叉点。
- 进将、进帅 :: 将或帅从肋线回到中路；
- 钓鱼马 :: 只处于对方正马位的马，这时存在底线进车叫杀的手段。类似的马位也一并称作钓鱼马位；
- 高兵 :: 指处于对方骑河线或者卒林线的兵；
- 高车 :: 可以平车打将的车位；
